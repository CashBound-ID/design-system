# Radius

Radius values define the rounding of corners in our design system. These values help maintain consistency across components, ensuring a unified and polished appearance.

### Retrieving Values
You can access the radius values using the `useDesignSystemProvider` hook:

```tsx
import { useDesignSystemProvider } from '@/context/DesignSystem';

const RadiusExample = () => {
  const { radius: { md } } = useDesignSystemProvider();

  return (
    <div style={{ borderRadius: md, padding: 16, background: '#eee' }}>
      This box has a medium border radius.
    </div>
  );
};
```
