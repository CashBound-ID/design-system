# Color

Our color principles built based on the recognition of cashbound brand colors to create not only consistent and usable.

### Retrieving Values
You can access the radius values using the `useDesignSystemProvider` hook:

```tsx
import { useDesignSystemProvider } from '@/context/DesignSystem';

const ColorExample = () => {
  const { color } } = useDesignSystemProvider();

  return (
    <div style={{ backgroundColor: color.VIOLET1200, padding: 16 }}>
      This box has a violet color.
    </div>
  );
};
```

These colors can basically include the need for color in background applications design.
